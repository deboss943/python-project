Python Questionnaire App

## Description
This is a simple Python terminal-based questionnaire application.  
It asks the user a series of questions, collects their responses, and displays a summary at the end.

This project is beginner-friendly and demonstrates basic Python concepts such as:
- Functions
- Lists
- Loops
- User input/output

---

## Features
- Asks multiple questions in the terminal
- Stores user responses
- Displays a summary of all answers
- Easy to modify or extend

---

## Requirements
- Python 3.x

---

## How to Run
1. Make sure Python is installed on your system.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the program using:

```bash
python questionnaire.py
